# Locale: Italian (Italy)

## Output language

**All recipe text MUST be written in Italian.**

This includes: title, ingredients, instructions, tips, tags, category, cuisine, and changes_made.

The JSON keys and field names are always in English — only the VALUES are in Italian.

## Italian measurements — CRITICAL

**Use metric weights (grams) as primary measurement for precision.**

Italian cooking traditionally uses:

- **g/kg** for all solid ingredients
- **ml/l** for liquids
- **Cucchiaio (cucchiaio)** and **cucchiaino (cucchiaino)** exist but vary by ingredient

### Measurement preference order

**Always prefer:** g/kg for solids, ml/l for liquids

Avoid vague spoon measures unless traditional (e.g., "un cucchiaio di olio").

### Practical rounding

Round to practical metric amounts:

| Impractical | → Practical |
| ----------- | ----------- |
| 37 g        | 40 g        |
| 123 ml      | 125 ml      |
| 87 g        | 90 g        |

### Traditional volume measures

Where traditional, use:

- **Bicchiere** (glass, ~200-250 ml) — specify ml in parentheses
- **Tazzina** (small cup, ~80 ml) — for espresso-sized portions

**Examples:**

- "1 bicchiere di vino (200 ml)"
- "1 tazzina di parmigiano grattugiato (80 g)"

### Vague amounts — Italian conversions

- "A pinch of salt" → "1 pizzico di sale" or "quanto basta (q.b.)" for flexible amounts
- Use "q.b." (quanto basta) only when precision isn't critical

## Italian dairy products — preserve product type

Use Italian dairy products, NOT foreign substitutions.

| Product type           | Italian equivalent                 | Notes                                         |
| ---------------------- | ---------------------------------- | --------------------------------------------- |
| Panna fresca/da cucina | Panna fresca (35-38% grassi)       | For cooking, not whipping                     |
| Panna da montare       | Panna da montare (35% grassi)      | For whipping                                  |
| Ricotta                | Ricotta vaccina                    | Standard cow's milk ricotta                   |
| Mascarpone             | Mascarpone                         | 40-50% fat, for desserts/risotto              |
| Yogurt greco           | Yogurt greco                       | Thick yogurt                                  |
| Panna acida            | Panna acida                        | Less common, use yogurt greco or panna fresca |
| Latte                  | Latte intero/parzialmente scremato | Specify type if relevant                      |

**For lactose-free:**

- Add "(senza lattosio)" after the ingredient
- "Panna fresca (senza lattosio)"
- "Latte (senza lattosio)"

## Italian cooking conventions

### Olive oil

- **Olio extravergine d'oliva** for finishing, salads, traditional dishes
- **Olio di oliva** or **olio di semi** for frying (when specified)
- Default assumption: use olio extravergine unless recipe specifies otherwise

### Tomatoes

Use Italian tomato product names:

- **Pomodori pelati** (whole peeled tomatoes)
- **Passata di pomodoro** (tomato purée)
- **Concentrato di pomodoro** (tomato paste)
- **Pomodorini** (cherry tomatoes)

### Pasta

- Always specify type: "Spaghetti", "Penne", "Rigatoni", etc.
- Standard portion: 80-100 g per person (dry weight)

### Broth/Stock

- **Brodo vegetale/di carne/di pollo** — homemade or from cube
- "Dado vegetale/di carne" for bouillon cubes

## Packages and containers — standard Italian sizes

ALWAYS replace "confezione", "lattina", "scatola" with actual measurements:

- "1 confezione di passata" → "500 g di passata di pomodoro"
- "1 scatola di pelati" → "400 g di pomodori pelati"
- "1 confezione di pasta" → "500 g di pasta" (or specify 250 g, 500 g)
- "1 lattina di tonno" → "80 g di tonno sott'olio"
- "1 confezione di panna" → "200 ml di panna fresca"

Standard Italian package sizes:

- Pasta: 500 g (or 250 g)
- Pelati/passata: 400-500 g
- Panna: 200 ml
- Mozzarella: 125 g
- Tonno: 80 g

## Italian ingredient availability

When suggesting alternatives or substitutions, use ingredients commonly found in Italian supermarkets (Esselunga, Coop, Conad, Carrefour):

- **Vegetarian protein**: Tofu, seitan, legumi — NOT heavily processed meat substitutes
- **Cheese**: Parmigiano Reggiano, Grana Padano, pecorino, mozzarella — use proper Italian names
- **Fresh herbs**: Basilico, prezzemolo, rosmarino, salvia, origano fresco
- **Pantry staples**: Use Italian product names (es. "farina 00" not "all-purpose flour")

## Portion markers

If source contains portion markers like `[X | Y]`:

**ALWAYS extract and use the 4-PORTION (4P) value:**

| Original                | → Convert to         |
| ----------------------- | -------------------- |
| `water [3 dl \| 6 dl]`  | 600 ml di acqua      |
| `salt [½ tsp \| 1 tsp]` | 1 cucchiaino di sale |
| `[1 package, 2P]`       | 2 confezioni         |

**Remove all portion markers from the final result.**
